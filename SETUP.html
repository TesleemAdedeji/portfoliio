📧 Contact Form Setup

To enable contact form functionality, choose one option:

Option 1: Formspree (Recommended)

  1. Go to formspree.io
  2. Sign up and create a new form
  3. Copy your form ID (e.g., f_xxxxx)
  4. Open index.html and find the contact form
  5. Add action: action="https://formspree.io/f/YOUR_ID"

Option 2: EmailJS

  1. Go to emailjs.com
  2. Sign up and configure email service
  3. Add EmailJS script to HTML head
  4. Initialize with your API key

Option 3: Discord Webhook

  1. Create Discord server and get webhook URL
  2. Add webhook URL to JavaScript code
  3. Form submissions will appear in Discord

🎨 Change Colors

Edit the CSS variables in css/styles.css (lines 1-15):

Current Primary Color: #00d4ff (Cyan)

🐛 Troubleshooting

Images not showing?

Solution: Check file paths. Images must be in assets/ folder with correct filenames (profile.jpg, project1.jpg, etc.)

Styles not applying?

Solution: Hard refresh browser (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac). Check CSS files are linked in HTML head.

Mobile menu not working?

Solution: Check browser console (F12) for JavaScript errors. Ensure main.js is linked correctly.

Form not submitting?

Solution: Setup email service integration (see Email Integration section above).
